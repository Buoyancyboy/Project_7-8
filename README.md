# Aquabots: Loggermodule

## Project layout

- Dockerfiles + docker-compose in `/`.
- SQLite database logger in `/data`.
- Mosquitto MQTT broker `configs in /mosquitto/config`.
- Django backend in `/src`.
- React in `/src/hmi_frontend`.
